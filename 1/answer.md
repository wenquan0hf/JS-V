打开控制台的方式:
- 鼠标在页面上右键，选择“审查元素”，然后切换到 Console 面板。
- Chrome 浏览器，右上角 工具(坚排的三个点图标) -> 更多工具 -> 开发者工具
- Fire 浏览器，右上角 工具(三条横线图片) -> Web 开发者 -> Web 控制台
- Chrome 浏览器，Ctrl + Alt + i
- Fire 浏览器，Ctrl + Alt + k
